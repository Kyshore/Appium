Install Appium

Installing Appium is as easy as running a single NPM command:

Microsoft Windows [Version 10.0.22621.2428]
(c) Microsoft Corporation. All rights reserved.

C:\Users\newky>npm install -g appium|

This command installs Appium globally on your system,
so that you can access it from the command line simply by running the appium command.
Go ahead and run it now:

Microsoft Windows [Version 10.0.22621.2428]
(c) Microsoft Corporation. All rights reserved.

C:\Users\newky>appium|

You should see some output that starts with a line like this:

Microsoft Windows [Version 10.0.22621.2428]
(c) Microsoft Corporation. All rights reserved.

C:\Users\newky>appium

[Appium] Welcome to Appium v1.22.3

That's it! If you get this kind of output, the Appium server is up and running.
Go ahead and quit it (CTRL-C) and move on to the next step, where we'll install a driver for automating Android apps.

The above command is used both for checking the Appium Version also to start the Appium Server.

When you installed Appium Version 1x, all available drivers would be
installed at the same time as the main Appium server.

Installing Appium 2.0 (e.g., by npm install -g appium
or
npm install --location=global appium),
will install the Appium server only, but no drivers.
To install drivers, use the following command

Microsoft Windows [Version 10.0.22621.2428]
(c) Microsoft Corporation. All rights reserved.

C:\Users\newky>appium driver install uiautomator2
